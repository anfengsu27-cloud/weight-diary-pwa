# 体重日记 - 减重助手

**编号：** project_001
**创建日期：** 2026-06-30
**描述：** 减肥体重记录与运动热量追踪的移动端 PWA 应用

## 功能

- **体重记录** — 每日记录体重，自动计算减重进度和趋势
- **运动追踪** — 内置 12 种常见运动类型，自动计算热量消耗
- **仪表盘** — 今日数据概览、最近记录、快速录入
- **趋势图表** — 体重变化折线图、运动消耗柱状图
- **进度环** — 减重进度和每日运动目标可视化
- **目标设定** — 目标体重、起始体重、日消耗目标、BMI 计算
- **数据管理** — 导出 / 导入 / 清除数据
- **PWA 支持** — 可添加到手机主屏幕，离线可用

## 使用方式

1. 用手机浏览器或电脑浏览器打开 `src/index.html`
2. 在浏览器菜单中选择「添加到主屏幕」即可像原生 App 一样使用
3. 所有数据存储在设备本地，无需联网

## 技术栈

- 纯 HTML5 + CSS3 + JavaScript（零外部依赖）
- Canvas 绘制图表（折线图、柱状图、进度环）
- localStorage 持久化数据
- PWA manifest + Service Worker 支持离线
- 响应式移动端优先设计

## 运动类型与热量计算

内置 12 种运动类型，采用 **MET 代谢当量** 公式自动计算：

```
消耗（kcal）= MET × 体重(kg) × 运动时间(小时)
```

支持自定义运动手动输入热量。

## 数据隐私

所有数据仅存储在你的浏览器本地（localStorage），不会发送到任何服务器。
支持导入 / 导出 JSON 备份。
