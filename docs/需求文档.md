# 体重日记 - 需求文档

**版本：** v1.0
**日期：** 2026-06-30

## 产品概述

一款用于记录减肥过程中体重变化和运动热量消耗的移动端应用。
目标是提供简洁、直观、完整的减重数据追踪体验。

## 功能清单

### P0 — 核心功能

- [x] 体重记录（日期、重量、备注）
- [x] 体重历史查看与编辑删除
- [x] 运动记录（类型、时长、自动/手动热量）
- [x] 运动历史查看与编辑删除
- [x] 首页仪表盘（今日概览）
- [x] 本地数据持久化（localStorage）

### P1 — 数据分析

- [x] 体重趋势折线图（7天/30天/全部）
- [x] 运动消耗柱状图（7天/30天/全部）
- [x] 减重进度环
- [x] 日运动目标进度环
- [x] 统计卡片（记录天数、最低体重、今日消耗）

### P2 — 个性化设置

- [x] 目标体重设定
- [x] 起始体重设定
- [x] 身高设定（BMI）
- [x] 日消耗目标设定
- [x] 昵称

### P3 — 数据管理

- [x] 导出数据（JSON）
- [x] 导入数据（JSON）
- [x] 清除所有数据
- [x] PWA 离线支持
- [x] 添加到主屏幕

## 技术决策

| 决策 | 选择 | 原因 |
|------|------|------|
| 技术方案 | PWA (纯前端) | 零服务器，零安装，跨平台 |
| 数据存储 | localStorage | 便捷、足够、无需后端 |
| 图表 | Canvas 纯手绘 | 零外部依赖 |
| 热量计算 | MET 公式 | 科学、可配置 |
